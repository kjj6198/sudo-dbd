# sudo dbd
* [demo](http://sdbd.herokuapp.com/)
:watch:
為了解決公司內部訂餐問題而製作的app，主要是希望能夠讓訂餐流程簡化，訂餐者只要發起點餐就可以，由程式內部計算總額、截止時間等，節省大家的溝通成本。
